## 💼 Featured Projects

### 🌿 **AyurvedaLearn** — Comprehensive BAMS Educational Platform
> **MERN Stack | TypeScript | Dark Mode | Bilingual (EN/MA)**

A full-scale learning platform designed specifically for **BAMS (Bachelor of Ayurvedic Medicine & Surgery)** students. This is not just a website—it's a knowledge base at W3Schools scale.

**What makes it special:**
- 📖 **Complete syllabus mapping** with interlinked topics
- 🌿 **Herb encyclopedia** with detailed properties & therapeutic uses
- 📊 **Progress tracking** with study analytics
- 🎨 **Dark mode** for comfortable late-night studying
- 🌐 **Bilingual support** (English & Marathi)
- 🔐 **User authentication** (JWT + bcrypt)

**Tech Stack:** MongoDB | Express | React + Vite | Node.js | TypeScript | Tailwind CSS

### 🏆 **Akshar Drishti** — Document Layout Detection for Multilingual OCR
> **YOLOv11 | Computer Vision | K.K. Wagh Institute Seminar Project**

Advanced document analysis system for **multilingual Indian OCR**, built with YOLOv11 for precise layout detection.

**What it does:**
- 🔍 Detects text regions, tables, and headers in documents
- 🌐 Handles multiple Indian scripts (Hindi, Marathi, Telugu, etc.)
- 📋 Prepares layout data for accurate OCR pipeline
- 📚 Complete literature reviews & dataset guides included

**Deliverables:** Literature reviews | Presentation scripts | Dataset guides | Flowcharts | Implementation
